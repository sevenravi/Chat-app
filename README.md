## Installation

1.Clone the repository:

```bash
   git clone https://github.com/sevenravi/Chat-app
```

2.Navigate to the project directory:

```bash
    cd ChatAPP
```
3.Install Dependencies:
```bash
    npm Install
```
4. Start the application:
```bash
    npm run dev
```


## Tech Stack

**Client:** React, Redux, TailwindCSS, NextUI

**Server:** Node, Express
